Anaconda使用

Posted on 2020-02-14 In 开发 , python Views: Valine:

下载源

  • Anaconda（包含大量科学计算的包）

    官网地址： https://www.anaconda.com/download/

    清华大学下载源： https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/

    Anaconda 镜像使用帮助： https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/

  • Miniconda（只包含python与conda，推荐使用）

    官网地址： https://conda.io/miniconda.html

    清华大学下载源： https://mirrors.tuna.tsinghua.edu.cn/anaconda/miniconda/

    安装过程截图

注意：anaconda安装路径务必不要出现空格，否则后面坑的不知东南西北

第一个勾：是是否把Anaconda加入环境变量，这涉及到能否直接在cmd中使用conda、jupyter、ipython等命令，推荐打勾，如果不打勾话问题也不大，可以在之后使用Anaconda提供的命令行工具进行操作。

第二个勾：是是否设置Anaconda所带的Python 3.6为系统默认的Python版本，这个自己看着办，问题不大。

安装完成后配置源

  • 添加国内的仓库镜像地址（官方源速度慢），执行以下命令即可：

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/

conda config --set show_channel_urls yes
